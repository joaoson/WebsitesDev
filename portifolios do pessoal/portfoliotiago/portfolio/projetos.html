TeamFinder

Tools: HTML, CSS and Photoshop

"TeamFinder" is a system idea to improve team building in on-line games. Its main purpose is to maximize chemistry between teammates, match compatible play styles and contribute to a better gaming experience.

The project was a group effort between me and some college colleagues and everything was coordinated using Kanban methodology and Trello. My responsibility was planning and developing the website's graphical interface.

Blackjack

Tools: HTML, CSS and JavaScript

Simplified version of the game "Blackjack". This project was developed with the intention of experimenting with Object-Oriented Programming and reviewing development concepts I learned through an extracurricular JavaScript course.

The project is still in development and my next step is to add graphical elements to the game, like cards and actions/animations.

Host Project

Soda Machine

Tool: Python

Soda machine simulator with functionalities for withdrawing and depositing money and goods (for the machine admin), purchasing products and returning change.

The project was a partial grade requirement and its objective was to explore fundamental concepts of programming, such as logical sequences (if/else) and iterations (while/for).
